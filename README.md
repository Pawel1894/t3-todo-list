## Install process

1. Install [nodejs](https://nodejs.org/en/download/)
2. Navigate project folder in command line
3. Run `npm install`
4. Set all required env variables
5. Run `npm start`


## Build process

1. Open project folder in command line
2. Set all required env variables
3. Run `npm run build`
